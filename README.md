# โปรแกรมคำนวณประมาณการ ภ.ง.ด.51

เครื่องมือคำนวณประมาณการภาษีเงินได้นิติบุคคลครึ่งปี (ภ.ง.ด.51) แบบไฟล์เดียว ทำงานในเบราว์เซอร์ทั้งหมด

## ความสามารถ
- รองรับ SME (อัตราขั้นบันได) และบริษัททั่วไป (20%)
- นำเข้างบทดลอง (Trial Balance) จาก Excel ได้ 3 งวด และเปรียบเทียบพร้อม % ของรายได้ และคอลัมน์ปัดหลักพัน ('000)
- ปรับปรุงทางภาษี: ค่าใช้จ่ายที่ไม่ให้ถือเป็นรายจ่าย, รายได้ที่ได้รับการยกเว้น, ขาดทุนสะสมยกมา (5 รอบบัญชี)
- ประเมินความเสี่ยงประมาณการขาดเกิน 25% (มาตรา 67 ตรี)
- จัดสรรผู้รับผิดชอบ (Preparer / Reviewer / Approver) และพิมพ์สรุปแบบ ภ.ง.ด.51
- ข้อมูลเก็บในเบราว์เซอร์ของผู้ใช้ (localStorage) ไม่ส่งขึ้นเซิร์ฟเวอร์

หมายเหตุ: เป็นเครื่องมือช่วยคำนวณเบื้องต้น โปรดตรวจสอบกับผู้สอบบัญชี/กรมสรรพากรก่อนยื่นจริง
